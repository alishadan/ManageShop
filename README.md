## 📦 Installation

### Prerequisites
- a c ompiler
- make

### Step-by-step Guide

```bash
# Clone the repository
git clone https://github.com/alishadan84/manageshop.git

# Navigate to the project directory
cd manageshop

# Run make
make

# Run the program
./build/app
```
